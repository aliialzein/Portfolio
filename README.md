## Local Development

Clone repository:

```bash
git clone https://github.com/aliialzein/Portfolio
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

Build:

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```
